# 加载图像 (元数据)

此节点从输入文件夹加载图像，并将其所有元数据（PNG 信息等）提取为 JSON 字符串。

> [!IMPORTANT]
> 与 ComfyUI 原生的 **Load Image** 节点（截至 v0.9.1 版本）不同，此节点会显式提取元数据并通过接口输出，这是配合 **Get Image Metadata** 节点使用的必要前提。

## 输入

- **image** (图像): 从 ComfyUI 输入目录中选择一张图像。

## 输出

- **image** (图像): 加载的图像。
- **mask** (遮罩): 图像的 Alpha 通道（如果存在）。
- **metadata** (元数据): 提取出的所有元数据，格式化为 JSON 字符串。

## 用法

此节点对于检查存储在图像文件中的生成参数或其他元数据非常有用。输出字符串可以传递给其他节点进行处理或显示。
