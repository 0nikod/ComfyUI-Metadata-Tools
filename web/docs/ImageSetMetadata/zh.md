# 设置图像元数据 (Set Image Metadata)

此节点将自定义元数据附加到图像张量（Tensor）上。这些元数据会随图像在工作流中传递，并可在稍后提取或保存到文件中。

## 输入

- **image** (图像): 要附加元数据的图像张量。
- **metadata** (元数据): 要附加的字符串或 JSON 格式的元数据。

## 输出

- **image** (图像): 内部属性中已附加元数据的图像张量。

## 用法

此节点对于在工作流中为图像标记自定义信息非常有用。这些信息随后可以被 **Save Image (Metadata)** 节点识别并保存到最终的 PNG 文件中。

> [!WARNING]
> 使用 ComfyUI 原生的 **Save Image** 节点会使嵌入的工作流和提示词覆盖掉由此节点附加的自定义元数据。为了在保存的文件中保留自定义元数据，您**必须**使用本插件提供的 **Save Image (Metadata)** 节点。
