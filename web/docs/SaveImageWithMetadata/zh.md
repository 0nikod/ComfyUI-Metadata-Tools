# 保存图像 (元数据)

此节点将图像保存到输出文件夹，同时嵌入作为字符串或 JSON 提供的自定义元数据。

## 输入

- **images** (图像): 要保存的图像。
- **metadata_text** (元数据文本): (可选) 包含要嵌入的元数据的字符串或 JSON 对象。如果留空，节点将尝试读取附加在输入图像张量上的元数据（例如来自 **Set Image Metadata** 或 **Load Image (Metadata)** 节点的元数据）。
- **filename_prefix** (文件名首缀): 保存文件的前缀。

## 功能

- **自动包含**: 如果可用，自动在 PNG 元数据中包含当前的工作流和提示词。
- **自定义元数据**: 允许将任何自定义文本或 JSON 数据添加到图像信息中。

## 用法

使用此节点保存最终结果，同时确保重要信息（如特定设置或标签）直接存储在图像文件中，以便将来参考。
